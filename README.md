## Twitter Clone
#### Project built with:-
-   PHP
-   Laravel
-   Javascript
-   Tailwindcss
-   Alpine js
-   Mysql Database

#### Project Features:-

-   Authentication and Authrization
-   Following
-   View your following tweets
-   Like or dislike Tweet
-   Edit your profile
-   Upload Avatar
